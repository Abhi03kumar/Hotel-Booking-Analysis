 Hotel Booking Data Analysis (EDA Project)

 Project Overview
This project focuses on performing Exploratory Data Analysis (EDA) on a hotel booking dataset using Python.  
The goal is to understand booking patterns, customer behavior, and cancellation trends.

---

 Tools & Technologies
- Python  
- Pandas  
- NumPy  
- Matplotlib  
- Seaborn  

---
 Key Insights
- Around 37% bookings were canceled  
- City hotels have higher cancellation rates  
- Peak bookings occur in July & August  
- Higher lead time increases cancellation probability  
- More guests lead to higher ADR (Average Daily Rate)  
- Customers with special requests are less likely to cancel  

---

 Analysis Performed
- Data Cleaning & Preprocessing  
- Feature Engineering  
- Univariate & Bivariate Analysis  
- Data Visualization (Countplot, Histogram, Heatmap, Scatterplot)  

---

 Project Objective
To analyze hotel booking data and extract meaningful insights that can help in better business decision-making.

---

 Dataset
Hotel Booking Dataset (Kaggle)
